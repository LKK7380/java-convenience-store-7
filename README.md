# 구현할 기능 목록

## 1. 도메인 구현
### Product 관련 기능
- [ ] 상품 기본 정보 관리
  - 이름, 가격, 수량, 프로모션명 포함
  - 유효하지 않은 값 검증
    - 이름이 null이거나 빈 값인 경우
    - 가격이 0 이하인 경우
    - 수량이 0 미만인 경우


- [ ] 프로모션 관련 기능
    - 프로모션 정보 관리
    - 프로모션 유효성 검증
- [ ] 주문 관련 기능
    - 주문 정보 관리
    - 주문 금액 계산

## 2. 파일 입출력
- [ ] products.md 파일 읽기
- [ ] promotions.md 파일 읽기

## 3. 사용자 입력 처리
- [ ] 상품 주문 입력
- [ ] 프로모션 관련 입력
- [ ] 멤버십 할인 관련 입력
- [ ] 추가 구매 관련 입력

## 4. 할인 계산
- [ ] 프로모션 할인
- [ ] 멤버십 할인
- [ ] 최종 금액 계산

## 5. 출력 기능
- [ ] 상품 목록 출력
- [ ] 영수증 출력
